# Football Leagues

A comprehensive Android mobile application designed to help users search, browse, and manage information about football clubs and leagues from around the world.

## Overview

Football Leagues is an Android application that provides detailed information about football clubs organized by league. Users can search for clubs, browse league information, and access comprehensive club details through both database and web searches.

## Features

- **League Management**: Add and manage football leagues in the local database
- **Club Search**: Search for clubs by name or league affiliation
- **Database Integration**: Local SQLite database for offline access
- **Web Integration**: Search clubs from web sources for real-time data
- **Club Details**: View comprehensive information about football clubs
- **User-Friendly Interface**: Intuitive navigation with material design
- **Multiple Search Options**: Search by club name or by league
- **Data Persistence**: Save league and club data locally

## Technologies

- **Language**: Kotlin
- **Frontend Framework**: Jetpack Compose
- **Database**: SQLite with Room (Android Jetpack)
- **Build System**: Gradle (Kotlin DSL)
- **UI Design**: Material 3 Design System
- **Network**: Internet connectivity support

## Getting Started

### Prerequisites

- Android Studio (Latest version)
- JDK 21+
- Android SDK 12 or higher

### Installation

1. Clone the repository:
```bash
git clone https://github.com/MZ-MUZNI/Football_Leagues.git
```

2. Open in Android Studio and build the project

3. Run on emulator or device

## How to Use

1. **Launch the Application**: Open the app on your Android device

2. **Add Leagues**:
   - Navigate to "Add Leagues DB" option
   - Enter league information
   - Save to local database

3. **Search by League**:
   - Select "Search Clubs by League"
   - Choose a league from the list
   - Browse clubs in that league

4. **Search by Club Name**:
   - Select "Search Clubs"
   - Enter club name
   - View search results

5. **Web Search**:
   - Select "Search Clubs from Web"
   - Enter search query
   - Access real-time information

## Requirements

- Android 12 (API level 31) or higher
- Internet permission for web search features
